# Dijkstra Algorithm

### 출발점에서 목표점까지의 ***최단 경로***를 구하는 알고리즘

## 특징
- 한 노드에서 다른 모든 노드로의 최단 경로를 구할 수 있음
- 단, edge에 0보다 작은 weight가 없어야 함
- **_Greedy + DP_**
- 복잡도: O(ElogV)
