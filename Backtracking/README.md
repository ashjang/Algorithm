# Backtracking

### : 모든 경우의 수를 탐색하며 최적해를 구하는 과정에서 해가 아니어서 막히면, 되돌아가서 다시 해를 찾는 방법. 즉, 유망하지 않으면 back!

 - promising: 해가 될 가능성 O
 - pruning: 해가 될 가능성 X
 - backtracking: 유망하지 않은 쪽으로 가지 않고 되돌아가기

### 특징
- 재귀함수로!
- 시간 단축
- 문제 속 조건을 잘 찾기
- DFS
